# 대장정 (大長征)

> 삼국(백제 vs 고구려) 소재의 **웹 전략 게임** 프로토타입.
> 순수 **HTML / JavaScript / Canvas** — 빌드 불필요, 서버 불필요. HTML 파일을 브라우저로 열면 바로 실행됩니다.

**컨셉:** 삼국지7(코에이)의 RTS화. 장수를 성에 주둔시켰다가 병력을 붙여 **출진**시키고, 지도 위에서 실시간으로 부대를 움직여 천하를 정벌하는 **총력전(`total.html`)이 메인 모드**입니다.

**핵심 철학:** 병력 수가 많다고 무조건 이기지 않는다. **소수라도 야습·매복·지형·사기 붕괴를 활용하면 대군을 격파**할 수 있어야 한다 — 이게 있어야 단순 숫자 싸움이 아닌 전략 게임이 된다.

---

## 실행 방법

설치·빌드가 필요 없습니다. 아래 파일을 **데스크톱 크롬으로 열면** 됩니다.

```
index.html   → 시작 메뉴 (여기서 각 모드 진입)
```

특정 모드를 바로 열려면 해당 HTML 파일을 직접 열어도 됩니다 (아래 표 참고).
로컬 `file://` 로 모든 스크립트가 로드되므로 별도 서버가 필요 없습니다.

### 헤드리스 테스트 (node 18+)

전투 코어는 DOM 없는 순수 모듈이라 화면 없이 숫자로 검증됩니다.

```bash
node test/verify.js        # engine.js(판정 라이브러리) 유닛 테스트
node test/war_sim.js       # warsim.js 실시간 전투 — 종료성/균형
node test/campaign_sim.js  # (stale) 옛 영지전 판정 검증
node test/tune.js          # (참고) 영지전 파라미터 스윕
```

> `total.html`이 가장 발전된 메인 모드이고, `campaign.html`은 그 이전의 간이 전략맵입니다.

### total.html 기본 조작
- **장수 클릭** = 선택
- **맵 클릭** = 진군(이동 명령)
- **아군 성 클릭** = 출진 편성 (선택 중이면 위황/재정비)
- **적 성 클릭** = 점령

스크립트 로드 순서:
`warfield` = engine→data→warsim · `total` = engine→data→warsim→rts · `campaign` = engine→data→warsim→glossary · `codex` = engine→data→glossary

## 핵심 시스템 (용어)

- **장수** — 부대의 대장. 무력·통솔·지력·민첩 4능력치. 병사 수백 명 몫의 영웅 유닛.
- **병종 상성** — 창병 ▶ 기병 ▶ 궁병 ▶ 창병 (가위바위보). 보병은 중립.
- **사기** — 부대 정신력. 낮으면 도주, 0이면 붕괴 → "소수 역전"의 엔진.
- **진형** — 쐐기/방진/횡대/종대. 대형 간 상성 존재.
- **지형** — 평야/언덕/숲/강/산/늪. 이동·전투 보정.
- **민심** — 성의 체력 겸 생산력. 0이면 함락.
- **공성전** — 포위 시 성벽 반격 + 보급 차단으로 서서히 함락.
- **장비** — 무기/갑옷/장갑/부츠 착용, 능력치 반영.

자세한 공식·튜닝 위치는 [HANDOVER.md](HANDOVER.md) 참고.
