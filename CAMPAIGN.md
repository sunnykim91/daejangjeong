# 영지전 규칙 명세서 (v0 · MVP)

> 🗄️ **[부분 참고용]** 이 영지전(`campaign.html`)은 여전히 동작하지만, **메인은 `total.html` 총력전**으로 이동함(HANDOVER §4-2). 영지전은 warfield 실전전투와 연결된 초기 전략 레이어. 밸런스 상수·민심 개념은 참고 가치 있음. 신규 개발은 total.html 중심.

> **목적**: 일기토(GAMERULE.md) 위에 얹는 **바깥 전략 레이어**. 지도 위 영지를 두고 출정·점령으로 영토를 다툰다.
> **범위(v0)**: 지도 + 민심 + 출정 + 점령 + 간단 적 AI + 승패. 내정(경제)·외교·계절·이벤트는 v0.1+에서 얹는다.
> **연결**: 일기토의 `BattleResult`(GAMERULE 9장)를 입력으로 받는다. 같은 시드 = 같은 결과 철학 유지.

## 1. 지도 / 영지 (Territory)

지도는 **영지 노드 그래프**다. v0은 6개.

| 영지 | 초기 소유 | 비고 |
|---|---|---|
| 위례성 | 백제(아군) | 아군 수도 |
| 웅진 | 백제(아군) | |
| 한성 | 백제(아군) | 전선 |
| 평양성 | 고구려(적군) | 적 수도 |
| 국내성 | 고구려(적군) | |
| 졸본 | 고구려(적군) | 전선 |

각 영지가 가진 값:
```
{ id, name, owner: 'player'|'enemy', minsim: 0~100, adj: [인접 영지 id…] }
```
- `minsim`(민심): **그 영지의 방어력 = 공성 진행도**. 초기값 60.
- `adj`: 인접 영지. **인접한 적 영지에만 출정 가능.**

> 〔왜 이렇게?〕 민심을 "영지의 체력"으로 삼아 한 시스템으로 공성을 표현한다. 별도 병력 수치 없이도 "여러 번 이겨야 함락"이 나온다. 그래프 인접은 전선을 만들어 "어디부터 칠까"라는 전략을 강제한다.

## 2. 턴 진행 (Campaign Loop)

```
1라운드 = 아군 1회 출정 → 적군 1회 출정.
1. (아군 턴) 내 영지 1곳 선택 → 인접 적 영지로 출정 → 일기토 → 결과 적용
2. (적군 턴) 적 AI: 자기 영지에 인접한 아군 영지 중 민심 가장 낮은 곳으로 출정 → 일기토 → 결과 적용
3. 승패 판정 → 다음 라운드
```

## 3. 출정 → 일기토 연결 (핵심)

출정하면 **일기토 엔진을 그대로 호출**한다. 단:

- **공격측 시작 사기 = 50** (GAMERULE 기본값)
- **방어측 시작 사기 = `round(20 + minsim × 0.6)`** — 민심 60→56, 85→71

> 〔왜 이렇게?〕 민심이 높은 영지는 수비군이 사기충천 → 잘 안 무너진다. 일기토 사기 시스템과 영지 민심을 연결해 두 레이어가 서로 영향을 주고받게 한다. 계수(0.6)는 시뮬레이션(test/tune.js)으로 "교착 없이 ~33라운드에 수렴"하도록 잡은 값.

### 3.1 결과 적용 (BattleResult → 민심)

공격측이 **이기면** 방어 영지 민심을 등급만큼 깎는다:

| 등급(grade) | 민심 감소 |
|---|---|
| 대승 | −55 |
| 승 | −38 |
| 신승 | −23 |

- 민심이 **0 이하 → 점령**: 소유권이 공격측으로 넘어가고, 민심을 **68**로 리셋(갓 점령했지만 어느 정도 굳음 — 즉시 재탈환 방지).
- 공격측이 **지면**: 방어 영지 민심 **+2**(수비 성공, 상한 85).

> 〔왜 이렇게?〕 등급 차등으로 "대승 한 방"의 가치를 만든다. 리셋 68은 핑퐁(점령→즉시 재탈환 반복) 교착을 없애려는 값. 이 수치들은 test/tune.js 스윕으로 결정했고, test/campaign_sim.js가 "300회 캠페인 전부 정상 종료(0 교착)"를 회귀 검증한다.

> 〔밸런스 메모〕 현재 AI vs AI 기준 아군 승률이 ~93%로 높다(주인공 보정 톤). 사람이 직접 지휘하면 더 높아짐 → **난이도는 v0.1 최우선 튜닝 대상**(적 로스터 강화 / 적 선공권 / 민심 결합 강도).

## 4. 승패 판정

- **승리**: 모든 적 영지를 점령(적 영지 0개).
- **패배**: 모든 아군 영지를 빼앗김(아군 영지 0개).

## 5. 적 AI (v0)

> 〔확장 훅〕 v0.1: 수도 우선 타격 / 병력 집중 / 후퇴 판단. GAMERULE 12장의 "통솔=AI 지능" 철학을 전략 레이어에도 적용.
